# College Web Project

## Used on the project

Backend: [Express](https://expressjs.com/)  
Front-end: [React](https://reactjs.org/) with [Chakra-UI](https://chakra-ui.com/)


## run servers

Install nodejs server libraries 
`npm install`

run node server  
`npm run dev`  


Install React Project libraries
`npm install`

run react server  
`npm run dev` on web directory
